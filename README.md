# CLI Todo

## How to setup on Linux

1. Clone the repository. 
2. Install `uv`: `curl -LsSf https://astral.sh/uv/install.sh | sh`.
3. Create virtual: `uv venv --python 3.14`
4. Activate virtual environment: `source .venv/bin/activate`. 
5. Run `python main.py` to start the app. 

## Features
1. **Add** a todo item with `add` command. 
2. **Edit** a todo item with `update` command. 
3. **Delete** a todo item with `delete` command. 
4. **View** existing items with `list` command. 
5. Enter `exit` command to quit the app.
